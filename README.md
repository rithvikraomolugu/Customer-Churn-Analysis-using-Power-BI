## Customer Churn Analysis using Power BI

This project analyzes customer churn behavior using Power BI and the Telco Customer Churn dataset.

The dashboard explores customer demographics, service subscriptions, contract types, and billing information to identify the key drivers of churn.

### Key Insights
- Customers with short tenure show the highest churn rate
- Month-to-month contracts have significantly higher churn
- Fiber optic internet users exhibit higher churn
- Customers without online security are more likely to churn

### Tools Used
- Power BI
- DAX
- Data Visualization
